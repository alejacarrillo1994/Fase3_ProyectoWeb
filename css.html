Actividades

¿QUE ES?

es un lenguaje que se utiliza para diseñar la presentacion del contenido de una pagina web a los usuarios.
el contenido de dicha pagina suele ser un archivo de texto estructurado utilziando el lenguaje de marcado
ya conocido como HTML. Existen otros archivos de texto tales como SVG o XML.


SINTAXIS BASICA CSS

Reglas:

Selectores de etiqueta: selecciona todos los elementos de la pagina donde la etiqueta HTML coincide con el valor del selector.
se muestra el siguiente ejemplo
Selectores de clase: aplica estilos a un solo elemento de la pagina, en donde solamente se pone el atributo class sobre el elemento al cual aplica directamente la regla css, para poder aplicarle el estilo css se debe referenciar con (.) y el nombre del valor.
selectores de ID: permite seleccionar un elemento de la pagina a traves del valor de su atributo id, este selector no permite en dos elementos diferentes de una misma pagina. Para poder aplicar el estilo css se debe referenciar con (#) y el nombre del valor.

Agrupamiento:

permite agrupar los selectores y declaraciones en una sola linea, esto evita que se repita tanto codigo innecesario de un atributo con declaraciones identicas.

Herencia:

permite heredar los valores de propiedades asignados al selector exterior. es decir todos los selectores que esten anidados dentro de selectores.
