HTML5

¿QUE ES?


HTML5 es un lenguaje markup (Hyper Text Markup Language) es la quinta version mas importante del lenguaje basico de html.
html5 especifica dos variantes de sintaxis para html: una clasica, html: (text/html),conocida como html5.
adicionalmente HTML5 establece nuevos elementos y atributos que reflejan el uso tipico de los sitios web modernos, algunos de ellos son similares a las etiquetas div y span pero tienen un significado semantico.

ETIQUETAS HTML5

Las etiquetas son la herramienta fundamental para que los navegadores puedan interpretar el codigo y permitirnos ver imagenes, texto, parrafo y estructuras, estas nuevas etiquetas nos ahorran el uso de otros productos que se usaban apra complementar y hacer cosaas que con el simple html no se podia hacer.
las etiquetas mas importantes de las nuevas etiquetas creadas son las siguientes:

  • Article: sirve para definir un articulo, un comentario de un usuario o una publicacion independiente dentro del sitio

  • Header, Footer: estas etiquetas individuales ahorran tener que insertar ids para cada uno, como solia ser antes,
    ahora se puede insertar headers y footers para cada seccion, en lugar de tener que hacerlo unicamente en general

  • Nav: esta etiqueta es un contenedor para enlaces de navegacion, donde proveen formas de acceder a las secciones del sitio web.

  • Section: esta etiqueta es una de las mas importantes de las novedades que trae HTML5, ya que se puede definir
    todo tipo de seccion dentro de un documento.

  • Audio y video: permite acceder de forma mas simple a contenidos multimedia que puede ser reproducido por casi todo tipo de dispositivos

  • Canvas: permite introducion un lienzo dentro de n documento, para poder dibujar graficos por vectores donde sera necesario
