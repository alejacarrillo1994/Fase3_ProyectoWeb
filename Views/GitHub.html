GITHUB

¿Que es?

Es una plataforma de desarrollo de software colaborativo para alojar los proyectos realizados,
utilizando el sistema de control de versionamiento llamado Git

¿Cuales son las ventajas?

  • Brinda herramientas muy utilices para trabajar los proyectos en equipo

  • facilita compartir codigo y editar de manera online

  • Manejo de versionamiento y backup seguro
